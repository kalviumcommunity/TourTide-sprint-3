# Scripts Directory

This directory contains utility scripts for the TourTide project.

## Purpose
- Store reusable code and utilities
- Automate data processing tasks
- Provide project-wide functionality

## Script Categories

### Data Processing Scripts
- `data_cleaner.py` - Clean and validate raw data
- `feature_engineer.py` - Create analysis features
- `data_validator.py` - Quality checks and validation

### Analysis Scripts
- `statistical_analysis.py` - Basic statistical operations
- `forecasting_model.py` - TourTide prediction models
- `visualization_utils.py` - Plotting and chart utilities

### Utility Scripts
- `config_manager.py` - Configuration and settings
- `logger.py` - Logging and debugging utilities
- `file_utils.py` - File I/O operations

## Usage Guidelines
- Scripts should be modular and reusable
- Include proper error handling
- Add comprehensive documentation
- Follow PEP 8 style guidelines

## Dependencies
- Document all required packages
- Use requirements.txt for specifications
- Include version compatibility information
