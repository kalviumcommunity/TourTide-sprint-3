# Raw Data Directory

This directory contains raw, unprocessed data files for the TourTide project.

## Purpose
- Store original datasets in their initial form
- Maintain data integrity and provenance
- Provide clean starting point for data processing

## Guidelines
- **Do not modify** files in this directory directly
- Create copies in `../processed/` for analysis
- Maintain original file names and formats
- Document data sources and acquisition dates

## Expected Contents
- Tourism datasets
- Historical visitor data
- External reference data
- Metadata and documentation files

## Data Sources
TourTide project will use tourism department data including:
- Visitor footfall records
- Seasonal tourism patterns
- Infrastructure utilization data
- Weather and event data
