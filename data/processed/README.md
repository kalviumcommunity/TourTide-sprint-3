# Processed Data Directory

This directory contains cleaned and processed data ready for analysis.

## Purpose
- Store transformed and cleaned datasets
- Maintain reproducible data processing workflows
- Provide analysis-ready data for notebooks

## Processing Pipeline
1. **Raw Data**: From `../raw/` directory
2. **Cleaning**: Remove duplicates, handle missing values
3. **Transformation**: Normalize formats, create features
4. **Validation**: Quality checks and consistency
5. **Storage**: Save processed data here

## File Naming Convention
- `dataset_cleaned.csv` - Cleaned version of raw data
- `dataset_features.csv` - Data with engineered features
- `dataset_final.csv` - Final analysis-ready data
- `processing_log.txt` - Documentation of transformations

## Expected Contents
- Cleaned tourism datasets
- Feature-engineered data
- Aggregated statistics
- Data quality reports
